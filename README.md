# MyEntraQuery - Enterprise AI Chat

A web application that integrates with Azure AI Foundry Agent (EnterpriseAgent) providing a chat interface with file upload capabilities, authenticated via Microsoft Entra ID.

## Features

- 💬 **Chat Interface** - Conversational interaction with EnterpriseAgent
- 📁 **File Upload** - Support for CSV, Excel, DOCX, and PDF files
- 🔐 **Entra ID Authentication** - Secure access via Microsoft identity
- 🌙 **Dark/Light Mode** - Toggle between themes
- 📋 **Copy to Clipboard** - Easily copy agent responses
- 📜 **Conversation History** - Persistent conversations across sessions
- 👥 **Multi-User** - Isolated conversations per authenticated user
